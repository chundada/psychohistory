## 🔥 核心技能速查（遇到问题先翻这几个）

### 当你要分析国际局势/冲突时

```
先看  ── 地缘政治三定律              → 定位博弈的基本规则
然后 ── 全球咽喉要道控制             → 找出关键节点
接着 ── 多重框架汇聚验证法 ⭐       → 看是否三方汇聚到同一判断
对照 ── 民族国家幻觉／伟人迷思      → 排除常见分析偏见
```

### 当你要预测社会趋势时

```
先看  ── 三维社会动力模型 (EOC)       → 评估社会生命力
然后 ── 帝国衰退三指标诊断法         → 检查衰退信号
接着 ── 精英过度生产与革命周期       → 是否存在结构性压力
对照 ── 相关即因果谬误              → 别把趋势当原因
```

### 当你在分析地缘战略时

```
先看  ── 非对称法则                  → 判断弱者vs强者
然后 ── 升级法则                    → 判断冲突会升级吗
接着 ── 邻近法则                    → 判断什么博弈最紧迫
深挖  ── 货币战争 / 美元瘾症预测     → 经济层面
对照 ── 帝国优势即劣势悖论          → 强者也有弱点
```

### 当你在研究文明/宗教现象时

```
先看  ── 末法汇聚预测法              → 识别末世论叙事
然后 ── 末世论作为地缘政治编码       → 解码宗教话语的真实意义
接着 ── Asabiyyah 边缘征服中心律     → 看谁有凝聚力优势
对照 ── 虚假二元对立                → 别掉进非黑即白
```

## 🧠 心理史学核心工作流

```
Step 1: 定位博弈
        用 地缘政治三定律 + 邻近法则
        问：这个局势有哪些玩家？最关键的博弈在哪一层？

Step 2: 评估力量
        用 非对称法则 + EOC 三维模型
        问：各方的能量、开放度、凝聚力如何？

Step 3: 检查信号
        用 帝国衰退三指标 + 精英过剩
        问：有没有结构性压力在积蓄？

Step 4: 解码叙事
        用 末法汇聚 + 末世论编码
        问：各方讲的"故事"是什么？是意识形态还是真实利益？

Step 5: 多重验证
        用 多重框架汇聚验证法 ⭐
        问：地缘、宗教、经济三个视角是否指向同一结论？

Step 6: 校准偏见
        用 7 个 Failure Mode Skills
        问：我在犯哪个常见错误？
```
