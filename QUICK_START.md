## 🎯 技能调用量级指南

> **调用多少 Skill 取决于问题复杂度，不是固定的。**

| 问题复杂度 | 示例场景 | 调用技能数 | 原因 |
|:-----------|:---------|:-----------|:-----|
| **简单** | 单一事实判断、术语查证 | **1-2 个** | 单框架足够，如术语词典+一个核心法 |
| **标准** | 地缘冲突分析、选举预测、宗教现象解码 | **3-5 个** | 需要 2-3 个独立框架交叉验证 + 1 个校准 |
| **复杂** | 长期趋势预测、文明兴衰诊断、大国博弈推演 | **5-7 个** | 需要 4-5 个框架 + 2 个校准+对比 |
| **高复杂度** | 战略级场景推演、跨文明级分析 | **6-8 个** | 全系列汇聚验证，但**不超过 8**——超出后注意力分散 |

> **经验法则**：宁少勿多。3 个高质量 Skill 的汇聚结论 > 7 个蜻蜓点水。
> 上下文有限时，优先选 **⭐ 标记的核心技能**。

## 🔥 核心技能速查（遇到问题先翻这几个）

### 当你要分析国际局势/冲突时

```
先看  ── 地缘政治三定律              → 定位博弈的基本规则
然后 ── 全球咽喉要道控制             → 找出关键节点
接着 ── 多重框架汇聚验证法 ⭐       → 看是否三方汇聚到同一判断
对照 ── 民族国家幻觉／伟人迷思      → 排除常见分析偏见
```

### 当你要预测社会趋势时

```
先看  ── 三维社会动力模型 (EOC)       → 评估社会生命力
然后 ── 帝国衰退三指标诊断法         → 检查衰退信号
接着 ── 精英过度生产与革命周期       → 是否存在结构性压力
对照 ── 相关即因果谬误              → 别把趋势当原因
```

### 当你在分析地缘战略时

```
先看  ── 非对称法则                  → 判断弱者vs强者
然后 ── 升级法则                    → 判断冲突会升级吗
接着 ── 邻近法则                    → 判断什么博弈最紧迫
深挖  ── 货币战争 / 美元瘾症预测     → 经济层面
对照 ── 帝国优势即劣势悖论          → 强者也有弱点
```

### 当你在研究文明/宗教现象时

```
先看  ── 末法汇聚预测法              → 识别末世论叙事
然后 ── 末世论作为地缘政治编码       → 解码宗教话语的真实意义
接着 ── Asabiyyah 边缘征服中心律     → 看谁有凝聚力优势
对照 ── 虚假二元对立                → 别掉进非黑即白
```

## 🧠 心理史学核心工作流（v9.0 七层汇聚验证）

### 框架选择：根据问题复杂度

| 问题复杂度 | 最少框架 | 推荐 | 包含的层 |
|:-----------|:---------|:-----|:---------|
| 简单事实判断 | 2 | 2 | ①事实基线 + ⑧校准 |
| 短期预测（1-3月） | 4 | 4 | ①③⑤⑧ |
| 中期趋势（半年-2年） | 6 | 6 | ①②③⑤⑥⑧ |
| 长期/文明级（3年+） | 8 | 全部 | ①②③④⑤⑥⑦⑧ |

### 执行流程

```
Step 1: 查询事实基线
        用 WebSearch 获取最新新闻/数据
        问：当前事实状态是什么？哪些是我以为真但已经变了的事？

Step 2: 识别历史模式    ← ②
        用 预测历史三要素 + 帝国衰退三指标
        问：历史上类似情况发生了什么？当前在模式的哪个阶段？

Step 3: 拆解博弈策略    ← ③
        用 非对称法则 + 升级法则 + 邻近法则
        问：各方的最邻近博弈是什么？谁在被迫升级？

Step 4: 诊断文明周期    ← ④
        用 精英过剩 + EOC模型 + 制度硬化
        问：社会是在上升、稳定还是衰落？内部压力有多大？

Step 5: 分析地缘战略    ← ⑤
        用 地缘三定律 + 咽喉要道 + 霍尔木兹杠杆
        问：地理条件如何约束各方策略？

Step 6: 追踪经济金融    ← ⑥
        用 货币战争 + 美元瘾症 + 债务周期
        问：美元霸权的健康度？哪些泡沫在膨胀？

Step 7: 解码宗教神话    ← ⑦
        用 末世论汇聚 + 卡巴拉加速主义 + 货币即神 + AI宗教
        问：各方在按什么叙事脚本行动？死亡焦虑在驱动什么？

Step 8: 校准偏见        ← ⑧
        用 虚假二元对立 + 伟人迷思 + 优势即劣势
        问：我是否在强迫汇聚？是否排除了第三种可能？

Step 9: 汇聚评分
        计算 同向框架数 / 总框架数
        完全汇聚(7/7) → 高置信度输出
        分歧 → 标记分歧层 → 诊断分歧原因 → 合成判断
```

### 汇聚评分标准

| 汇聚度 | 分数 | 含义 |
|:-------|:-----|:------|
| 完全汇聚 | 7/7 (100%) | 高置信度，输出核心预测 |
| 强汇聚 | 6/7 (86%) | 输出预测 + 标记分歧层 |
| 中等汇聚 | 5/7 (71%) | 输出预测 + 分析分歧原因 |
| 弱汇聚 | 4/7 (57%) | 只输出情景范围 |
| 发散 | ≤3/7 (<43%) | 不合成，只列各框架分析 |
