# 心理史学 | Psychohistory — 知识蒸馏系统设计规范

## 1. 概述

将 YouTube 频道 `@PredictiveHistory` 的 7 大系列（~160 集视频）蒸馏为可调用的 AI Skill 体系，命名为 **Psychohistory（心理史学）**。

### 设计目标

- 把博弈论、地缘政治、文明规律、宗教历史、时局分析提炼为方法论 Skill
- 先一次性蒸馏已有积累（Game Theory 为试点），再持续注入新内容
- 所有输出用中文，适配 Claude Code / Cursor Agent 调用

## 2. 核心架构：按系列为"书"

### 为什么按系列而非单集

| 维度 | 单集独立蒸馏 | 按系列聚合 |
|---|---|---|
| 方法论密度 | 低，每集只讲 1-2 个框架 | 高，系列内重复出现强化 |
| 跨集链接 | 需手动合并 | 自动发现隐性脉络 |
| 三重验证通过率 | 低（跨域证据不足） | 高（跨集证据丰富） |
| 冗余管理 | 大量重复 SKill | 去重合并 |

### 两阶段精炼流程

```
Phase 1: 逐集精提（轻量）
  ┌─ GT#1  →  extract_episode() → 框架 + 关键论断
  ├─ GT#2  →  extract_episode() → 框架 + 关键论断
  ├─ ...
  └─ GT#29 →  extract_episode() → 框架 + 关键论断

Phase 2: 跨集聚合（全量融合）
  29 集精提摘要 → 5路并行提取器 → 三重验证 → RIA++ → 产出
```

## 3. 7 路自定义提取器

| 编号 | 名称 | 提取内容 | 触发场景 |
|---|---|---|---|
| E1 | 博弈模型提取器 | 囚徒困境、胆小鬼博弈、信号博弈、承诺机制等框架 | 分析中有玩家/策略/收益/均衡 |
| E2 | 地缘法则提取器 | 心脏地带、修昔底德陷阱、海权vs陆权、缓冲区理论 | 讨论国家/帝国/地理/战争 |
| E3 | 文明规律提取器 | 精英过剩、人口转型、体制僵化、文明兴衰周期 | 讨论社会/文明/组织/制度 |
| E4 | 宗教叙事提取器 | 弥赛亚叙事、千禧年主义、一神教革命、神话结构 | 讨论宗教/信仰/神话/神圣文本 |
| E5 | 预测模型提取器 ⭐ | 信号→推演→结果链、条件依赖性预测、拐点识别 | 作者做出预测或推演未来 |
| E6 | 反例陷阱提取器 | 归因错误、确认偏误、过度简化、类比滥用 | 作者批评某种思维或分析错误 |
| E7 | 术语词典提取器 | 核心概念、作者独创术语、关键定义 | - |

## 4. 质量保障体系

### 四重验证（比原始多一重）

| 验证 | 核心问题 | 淘汰条件 |
|---|---|---|
| V1 跨域 | 至少 2 集有佐证？ | 只在一集出现 → 降级 |
| V2 预测力 | 能推导出未明说的结论？ | 纯描述无推断力 → 淘汰 |
| V3 独特性 | 不是常识？ | 任何人都能想到 → 淘汰 |
| **V4 体系兼容** | 与心理史学其他系列不矛盾？ | 与已建立的 SKill 冲突 → 打标 |

### 通过率预期

- 原始提取池：~100-200 个候选
- 三重验证后：~25-50 个（25-50%）
- 最终产出 SKill：~15-30 个

## 5. Pilot 范围：Game Theory 系列 (29集)

```
优先级最高，预计产出 15-25 个 SKill
内容覆盖：
  - 博弈论基础框架（GT#1-#10）
  - 地缘政治博弈（GT#11-#20）
  - AI / 未来预测（GT#21-#29）
```

## 6. 后续扩展路径

```
Game Theory (Pilot) → Civilization (59集) → Secret History (27集)
  → Geo-Strategy (19集) → Great Books + Dante (25集)
  → 新视频持续注入（按系列归入 → 增量蒸馏）
```

## 7. 技术栈

- **YouTube 音频下载**: yt-dlp
- **本地 ASR 转录**: Intel AIPC 本地模型
- **字幕直取**: yt-dlp --write-auto-sub （首选，快）
- **蒸馏引擎**: 基于 cangjie-skill RIA-TV++ 流水线
- **输出格式**: 兼容 Claude Code `~/.claude/skills/` / `.cursor/skills/`
