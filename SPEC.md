# 心理史学 | Psychohistory — 知识蒸馏系统设计规范

## 1. 概述

将 YouTube 频道 `@PredictiveHistory` 的 7 大系列（~160 集视频）蒸馏为可调用的 AI Skill 体系，命名为 **Psychohistory（心理史学）**。

### 设计目标

- 把博弈论、地缘政治、文明规律、宗教历史、时局分析提炼为方法论 Skill
- 先一次性蒸馏已有积累（Game Theory 为试点），再持续注入新内容
- 所有输出用中文，适配 Claude Code / Cursor Agent 调用

## 2. 核心架构：检索式提取（Retrieval-based Extraction）

### 核心设计理念

> **不用摘要取代原文，让提取器直接检索原文。**

29 集完整 VTT 字幕（~12MB 纯文本）保持原始精度，不经过任何中间摘要压缩。每个提取器以**检索信号（Search Signals）** 定位相关段落，直接在完整上下文中提取方法论。

### 流程

```
VTT 字幕全文（29集，零压缩）
       │
       ▼
┌──────────────────────────────────────────────┐
│  检索式提取（单阶段，不分 Phase）              │
│                                               │
│  每个提取器独立运行一次：                       │
│  1. 扫描全文中该领域专属的检索信号               │
│  2. 定位到信号段落后读取完整上下文                │
│  3. 从原文直接剪裁出方法论单元                   │
│  4. 每个提取结果包含 [source] = 原文精确引用      │
│                                               │
│  ┌──────────── ─────────  ──────────────┐     │
│  │ E1 博弈模型 │ E2 地缘法则 │ E3 文明规律  │     │
│  │ E4 宗教叙事 │ E5 预测模型 │ E6 反例陷阱  │     │
│  │ E7 术语词典                               │     │
│  └──────────────────────────────────────────┘     │
└──────────────────────────────────────────────┘
       │
       ▼
  candidates/（每个提取器一个子目录）
       │
       ▼
  四重验证 → RIA++ → 跨系列链接 → 压力测试 → 交付
```

## 3. 7 路检索式提取器

每个提取器不再"总结"或"概括"，而是**带着检索信号去原文定位**。

| 编号 | 名称 | 检索信号（Search Signals） | 输出类型 |
|---|---|---|---|
| E1 | 博弈模型提取器 | 囚徒困境、胆小鬼、信号博弈、承诺、均衡、MAD、升级、不对称、近距法则 | `framework` |
| E2 | 地缘法则提取器 | 心脏地带、修昔底德陷阱、海权、陆权、缓冲区、帝国、大国博弈 | `principle` |
| E3 | 文明规律提取器 | 精英过剩、人口转型、制度僵化、崩溃、兴衰、Asabiyyah、凝聚力 | `principle` |
| E4 | 宗教叙事提取器 | 弥赛亚、千禧年、末世论、一神教、神圣文本、预言、神话 | `case` / `principle` |
| E5 | 预测模型提取器 ⭐ | 如果…那么…、未来、将会、预测、推演、情景、拐点、信号 | `framework` |
| E6 | 反例陷阱提取器 | 错误、误解、陷阱、谬误、偏见、误读、不要… | `counter-example` |
| E7 | 术语词典提取器 | 作者自创术语、反复出现的核心概念、关键定义 | `glossary` |

### 检索信号使用规则

- 每个提取器独立扫描全文，不通读
- 检索信号命中时，读取前后至少 5 行上下文（或到自然段落结束）
- 提取结果必须包含 `[source] = 文件名 + 时间戳范围` 作为引文锚点
- 每个提取结果保持原文语言的原始措辞（英文），后附中文翻译/解读
- 宁可过度提取（多抓）也不漏掉——四重验证会做淘汰

## 4. 质量保障体系

### 四重验证

| 验证 | 核心问题 | 淘汰条件 |
|---|---|---|
| V1 跨域 | 至少 2 集有佐证？ | 只在一集出现 → 降级 |
| V2 预测力 | 能推导出未明说的结论？ | 纯描述无推断力 → 淘汰 |
| V3 独特性 | 不是常识？ | 任何人都能想到 → 淘汰 |
| **V4 体系兼容** | 与心理史学其他系列不矛盾？ | 与已建立的 Skill 冲突 → 打标 |

### 通过率预期

- 原始提取池：~100-200 个候选
- 四重验证后：~25-50 个（25-50%）
- 最终产出 Skill：~15-30 个

## 5. Pilot 范围：Game Theory 系列 (29集)

```
优先级最高，预计产出 15-25 个 Skill
内容覆盖：
  - 博弈论基础框架（GT#1-#10）
  - 地缘政治博弈（GT#11-#20）
  - AI / 未来预测（GT#21-#29）
```

## 6. 后续扩展路径

```
Game Theory (Pilot) → Civilization (59集) → Secret History (27集)
  → Geo-Strategy (19集) → Great Books + Dante (25集)
  → 新视频持续注入（按系列归入 → 检索式增量提取 → 重新链接）
```

## 7. 技术栈

- **YouTube 字幕下载**: yt-dlp `--write-auto-subs`
- **蒸馏引擎**: 检索式提取（7 路信号检索 + 上下文剪裁）
- **输出格式**: 兼容 `~/.claude/skills/Psychohistory/` / `.cursor/skills/Psychohistory/`
