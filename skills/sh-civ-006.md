## I — Interpretation（方法论解读）

青铜时代崩溃（约公元前 1200 年）是文明崩溃的经典案例：

**系统的脆弱性结构**：
1. **高度专业化**：每个城邦/王国依赖少数关键资源（锡、铜）的远距离贸易
2. **单点故障**：只要贸易链条中的一个节点断裂，整个系统瘫痪
3. **资本集中化**：财富集中在宫殿和神庙，普通人口生存储备薄弱
4. **外部扰动乘数**："海上民族"的入侵 → 贸易中断 → 饥荒 → 社会崩溃

崩溃连锁反应：
贸易中断 → 铜/锡短缺 → 武器减产 → 防御弱化 → 更多入侵 → 农业荒废 → 人口饥荒 → 宫殿经济崩塌 → 文字/技术失传 → 黑暗时代

核心教训：文明的复杂性与脆弱性成正比。

## A — Application（应用与执行）

**触发条件**：当分析全球供应链脆弱性或系统风险时应使用这个技能进行压力测试。

**作者案例**：Jiang 在 SH16 中用青铜时代崩溃的模型类比现代全球化供应链的脆弱性。

**执行步骤**：
1. 识别关键依赖：系统依赖哪些不可替代的单一资源或节点
2. 绘制贸易网络：绘制供应链接，标记单点故障位置
3. 计算系统韧性：冗余度、替代方案和储备水平
4. 模拟中断冲击：测试关键节点失效后的连锁反应
5. 评估崩溃概率：当 3 个以上关键节点处于高风险状态时 → 系统处于崩溃边缘

---

## B — Boundary（边界条件）

- 现代全球化的多样性和冗余度远高于青铜时代，直接类比需谨慎
- 并非所有崩溃都是系统性的——外部征服可能只是政体更替
- 此框架不能预测具体崩溃时间，只能评估系统的脆弱性程度
