# Stage 6: 七层汇聚验证（Convergence Verification）

> **心理学第七阶段**：将多重独立分析框架的结果汇聚对比——当多数框架指向同一方向时，置信度指数级提升。

---

## 1. 核心原理

**七层汇聚验证**是心理史学的最终分析步骤。它不是一个新的分析框架，而是一个**元分析方法**——将所有独立框架的输出汇聚在一起，判断它们是否指向同一结论。

### 为什么要汇聚？

| 单个框架 | 汇聚后的优势 |
|----------|-------------|
| 可能有系统性偏差 | 多层独立偏差不可能同时指向同一方向 |
| 有特定的盲区 | 不同框架覆盖不同的盲区 |
| 容易被观点带偏 | 多框架强制你从不同角度审视 |
| 无法自我验证 | 框架A的结论由框架B/C/D验证 |

### 核心假设

**如果七个相互独立的分析框架都指向同一结论，那么这个结论正确的概率远高于单个框架结论的概率。** 不是因为命运或天意，而是因为七个不同方向的误差恰好一致的概率极低。

## 3. 分层说明

### 第①层：事实基线（Factual Baseline）

**目的**：确保分析建立在真实世界的最新信息上，而非过时的假设。

**方法**：
- 用 WebSearch 获取最近 3-6 个月的新闻和数据
- 特别关注：价格数据、官方声明、重大事件、冲突升级/降级信号
- 标注"已确认事实"和"待验证信息"的区分

**关键问题**：
- "当前事实状态是什么？有什么新的变化？"
- "哪些是我以为是真的但实际上已经变了的事情？"

### 第②层：历史模式（Historical Patterns）

**目的**：从历史中找"押韵"——当前局势与哪些历史先例相似？走的是哪个模式？

**核心技能**：
- `sh-pre-003` — 预测历史三要素（模式+博弈+利益）
- `gt-predictive-005` — 帝国衰退三指标诊断法
- `gt-predictive-015` — 内战投射理论
- `interview-pred-causal-chain` — 因果链预测模型
- `gt-predictive-010` — 历史类比预测法

**关键问题**：
- "历史上类似的情况发生了什么？"
- "当前处于这个模式的哪个阶段？"

### 第③层：博弈策略（Game Strategy）

**目的**：拆解各方的利益计算、策略选择和互动结构。

**核心技能**：
- `gt-game-theory-asymmetry` — 非对称法则
- `gt-game-theory-escalation` — 升级法则（校准 vs 压制）
- `gt-game-theory-proximity` — 邻近法则（多博弈嵌套中的优先级）
- `sh-game-007` — 博弈论核心公理
- `interview-geo-hormuz-leverage` — 霍尔木兹不对称杠杆

**关键问题**：
- "各方的最邻近博弈是什么？"
- "谁在被迫升级？谁有不对称灵活性？"

### 第④层：文明周期（Civilization Cycles）

**目的**：从文明级的长期结构判断社会/制度是在上升、稳定还是衰落。

**核心技能**：
- `ph-origin-elite-overproduction` — 精英过剩模型
- `gt-civilization-eoc-model` — 三维社会动力模型（能量/开放/凝聚力）
- `gt-civilization-asabiyyah` — 阿萨比亚/边缘征服中心律
- `gt-civilization-institutional-sclerosis` — 制度硬化
- `gt-civilization-fertility-leading-indicator` — 生育率先行指标

**关键问题**：
- "社会的精英过剩程度如何？"
- "制度有没有硬化的迹象？"
- "社会凝聚力的趋势是上升还是下降？"

### 第⑤层：地缘战略（Geo-Strategy）

**目的**：分析地理约束、资源争夺和全球力量格局。

**核心技能**：
- `gt-geopolitics-three-laws` — 地缘政治三定律
- `gt-geopolitics-chokepoint` — 全球咽喉要道控制
- `interview-geo-hormuz-leverage` — 霍尔木兹海峡杠杆
- `gs-geopolitics-*` — Geo-Strategy 系列相关技能

**关键问题**：
- "谁控制着哪些关键节点？"
- "地理条件如何约束各方策略？"

### 第⑥层：经济金融（Economy/Finance）

**目的**：追踪货币、债务、资本流动和泡沫的结构性变化。

**核心技能**：
- `gt-geopolitics-currency-war` — 货币战争——美元霸权
- `gt-predictive-008` — 美元瘾症与崩溃预测
- `gt-religion-money-as-god` — 货币作为上帝（资本的神学结构）
- `interview-pred-2027-collapse` — 2027经济崩溃预测框架

**关键问题**：
- "美元霸权的健康度如何？"
- "哪些泡沫正在膨胀？"
- "资本流向是维持还是颠覆现有秩序？"

### 第⑦层：宗教神话（Religion/Mythology）

**目的**：解码宗教叙事、末世论和神秘学如何塑造行动者的决策框架。

**核心技能**：
- `gt-religion-eschatological-convergence` — 七大末世论汇聚
- `gt-religion-eschatology-geopolitics` — 末世论作为地缘政治编码
- `gt-religion-kabbalistic-redemption` — 卡巴拉通过罪恶救赎
- `gt-religion-messianic-accelerationism` — 弥赛亚加速主义
- `gt-religion-ai-stargate-rapture` — AI作为宗教项目
- `gt-religion-money-as-god` — 货币作为上帝
- `gs-rel-001` — 基督徒锡安主义的末世论引擎
- `civ-rel-012` — 吉尔伽美什的不朽追求（死亡焦虑）
- `gs-rel-006` — 末世论汇聚（地缘政治层面）
- `gs-rel-010` — 圣战与十字军——冲突的神学语言

**关键问题**：
- "各方相信什么叙事？他们在按什么脚本行动？"

### 第⑧层：校准层（Calibration / Failure Mode Check）

**目的**：在输出预测前，用思维陷阱清单检查是否存在系统性偏差。

**核心技能**：
- `gt-failure-false-dichotomy` — 虚假二元对立
- `gt-failure-great-man` — 伟人迷思
- `gt-failure-advantage-paradox` — 优势即劣势悖论
- `gt-failure-correlation-causation` — 相关即因果
- `gt-failure-dunning-kruger` — 达克效应
- `gt-failure-nation-state` — 民族国家幻觉
- `gt-failure-ai-consciousness` — AI意识幻觉
- `interview-fail-sunk-cost` — 沉没成本谬误

**检查清单**：
```markdown
☐ 我是否在强迫结果汇聚？（如果分的太多就承认"不确定"）
☐ 我是否把"可能"说成了"必然"？（条件式表达）
☐ 我是否排除了"非此即彼"的陷阱？（至少考虑3种情景）
☐ 我是否过度归因于个人而非结构？（邻近法则）
☐ 我是否用了最新的事实数据而非过时假设？
```

## 4. 汇聚评分机制

每个框架独立给出方向判断后，计算汇聚强度：

| 汇聚度 | 分数 | 含义 | 输出方式 |
|--------|------|------|----------|
| **完全汇聚** | 7/7 (100%) | 所有框架指向同一方向 | 输出"核心预测"，高置信度 |
| **强汇聚** | 6/7 (86%) | 一个框架有分歧 | 输出预测 + 标记分歧层 |
| **中等汇聚** | 5/7 (71%) | 两个框架有分歧 | 输出预测 + 分析分歧原因 |
| **弱汇聚** | 4/7 (57%) | 平分秋色 | 不输出强预测，输出情景范围 |
| **发散** | ≤3/7 (<43%) | 多数框架不一致 | 只输出各框架分析，不合成 |

### 分歧诊断流程

当框架出现分歧时，按以下流程诊断：

1. **标记分歧的层**：哪几个层与多数不一致？
2. **分析分歧来源**：
   - 数据层分歧？→ 重新验证事实
   - 框架适用性？→ 该框架是否适用于这个具体问题？
   - 时间尺度差异？→ 有的框架看短期，有的看长期
3. **判断分歧性质**：
   - **可调和分歧**：分歧来自不同时间尺度（如"短期涨长期跌"）
   - **不可调和分歧**：逻辑上相互矛盾
4. **合成判断**：
   - 如果是时间尺度分歧 → "短期X，但长期Y"
   - 如果是框架适用性 → "按多数框架，答案是X，但注意框架A因为Y原因可能不适用"
   - 如果不可调和 → 诚实说"框架之间存在根本性分歧"

## 6. 关键机制：为什么七个框架？

### 非冗余性保证

七个框架覆盖的维度**尽可能正交**（独立且不重叠）：

| 框架 | 盲区 | 由另一个框架覆盖 |
|------|------|------------------|
| 历史模式 | 未来可能有新变量 | 宗教神话（捕捉"新叙事"） |
| 博弈策略 | 忽略文化/信仰驱动力 | 宗教神话 |
| 文明周期 | 短期事件可能打乱长期趋势 | 事实基线 |
| 地缘战略 | 忽略内部精英斗争 | 文明周期（精英过剩） |
| 经济金融 | 认为人是理性的 | 博弈策略（认知局限） |
| 宗教神话 | 过于抽象，难以量化 | 经济金融（数据锚定） |
| 事实基线 | 无分析深度 | 所有上层框架 |

### 框架选择的灵活性

并非所有问题都需要7个框架。选择规则：

| 问题类型 | 最少需要 | 推荐 |
|----------|----------|------|
| 简单事实判断 | ①事实基线 + ⑧校准 | ①⑧ |
| 短期事件预测（1-3个月） | ①③⑤⑧ | 4个 |
| 中期趋势（半年-2年） | ①②③⑤⑥⑧ | 6个 |
| 长期趋势（3-10年） | ①②③④⑤⑥⑦⑧ | 全部 |
| 文明级推演（10年+） | ①②③④⑤⑥⑦⑧ | 全部 |

## 7. 输出格式

汇聚验证后的输出应遵循以下格式：

```markdown
## 分析结论

### 汇聚验证得分
- ⑦宗教神话层：方向A（证据...）
- ⑥经济金融层：方向A（证据...）
- ⑤地缘战略层：方向A（证据...）
- ④文明周期层：方向B ← 分歧（标记原因）
- ③博弈策略层：方向A
- ②历史模式层：方向A
- ①事实基线层：方向A

**汇聚强度**：6/7（强汇聚）

### 分歧诊断
第④层（文明周期）与多数不一致，原因：
- 时间尺度差异：文明周期看的是10年+趋势，其他框架看的是2-3年
- 合成判断：中期方向A，但长期有转向B的潜力

### 核心预测
[条件式预测 + 置信度 + 边界条件]

### 校准检查
☐ 虚假二分？已排除
☐ 伟人迷思？已排除
☐ 过度自信？已排除（标注了边界条件）
```
