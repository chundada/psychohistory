# Psychohistory Origin · 心理史学起源模块

> *"Psychohistory is the idea that if you can mathematically model history and it's a science... if it can be predicted then it can be controlled and harnessed for the betterment of humanity."*
> — Jiang Xueqin, Geo-Strategy Final Class

---

## 模块概述

**Psychohistory Origin** 是整个心理史学项目的**起源与哲学根基**模块。它来源于江学勤老师在地缘战略课最后一课中对心理史学的系统阐述——这也是用户启动这个项目的**原始灵感来源**。

这个模块不包含具体的地缘政治分析或历史案例，而是回答最根本的问题：
- 心理史学是什么？
- 为什么要做心理史学？
- 心理史学的方法论边界在哪里？
- AI 在心理史学中扮演什么角色？
- 人类在心理史学中扮演什么角色？

## 方法论地位

本模块在整个心理史学项目中处于**元方法论**的位置——它不提供具体的分析工具，而是提供：

1. **本体论**：心理史学是什么、不是什么
2. **目的论**：我们为什么要做这件事
3. **认识论**：什么是有效的心理史学知识
4. **价值论**：心理史学应该服务于什么目的

所有其他模块的技能，最终都应该能回溯到本模块的某个根本原理上。
