# MOC - 心理史学总览

> *"心理史学（Psychohistory）是一门介于心理学和历史学之间的学科，它使用数学公式来预测整个人类社会的未来行为。"*
> — Isaac Asimov，《基地》系列

---

## 项目概览

心理史学（Psychohistory）项目致力于将 YouTube 频道 `@PredictiveHistory` 的知识体系 —— 涵盖博弈论、地缘政治、文明分析、宗教历史与未来预测 —— **蒸馏为 AI 可调用的方法论 Skill**。

本项目从 Asimov 的《基地》系列获得灵感，但实践路径不同：我们不建模数学公式，而是从 160+ 集视频中**检索式提取可复用的思维框架**，经过四重验证后构造为结构化的 AI Skill。

## 知识蒸馏流水线

```
Stage 0: 系列理解 → 列出剧集 → 分组主题弧 → 识别核心论点
Stage 1: ⭐ 检索式提取 → 7路信号检索 → 定位原文 → 上下文剪裁
Stage 1.5: 四重验证 → V1跨域/V2预测力/V3独特性/V4体系兼容
Stage 2: RIA++ 构造 → R(原文)→I(方法论)→A1(案例)→A2(触发)→E(步骤)→B(边界)
Stage 3: 跨系列链接 → Zettelkasten 式网状连接
Stage 4: 压力测试 → 多场景交叉验证
Stage 5: 交付 → 部署到 skills 目录
```
