# 核心概念与术语表提取器 (Glossary Extractor)

## 用途 (Purpose)

从视频转录文本中提取**核心概念、作者自创术语、关键定义、反复出现的词汇**，构建该视频系列的领域专用术语表。这一提取器捕捉的是其他六个提取器无法涵盖的"元语言"——即主讲人特有的分析词汇、重新定义的日常用语、以及串联整个知识体系的枢纽概念。

## 范围 (Scope)

### 属于此提取器的内容
- **主讲人自创或显著改造的术语**：为该频道或特定系列独创的概念名称
- **关键定义**：主讲人对某个重要概念给出的显式或隐式定义
- **领域术语的独特用法**：日常词汇被赋予特定分析含义的使用方式
- **反复出现的核心概念枢纽**：在不同视频中反复出现、串联多个话题的概念
- **缩写和简称**：主讲人使用的特定的缩写、简称、代号
- **冲突或对立的术语对**：成对出现的对立概念（如 软实力/硬实力、陆权/海权）
- **隐喻和类比框架的术语**：主讲人使用的核心隐喻的术语表达
- **概念层次关系**：上位概念与下位概念的层级关系
- **年代/时期的特有名称**：主讲人用于指代特定历史时期的术语

### 不属于此提取器的内容
- 不需要解释的通用词汇（如"国家"、"战争"、"经济"的基础义）
- 仅在一期视频中出现一次且不重要的专有名词
- 不涉及分析框架的人名、地名、条约名称（除非有特殊概念价值）
- 对正文提取器（01-06）中已经详细记录的概念的简单重复
- 过于专业但在视频中只是背景提及的学科术语（如物理、化学概念）

## 识别信号 (Identification Signals)

在文本中寻找以下模式和关键词：

### 关键词与短语
```
"我称之为……"、"我们将它称作……"、"可以称为……"
"定义"、"定义为"、"意思是"、"指的是"、"即……"
"换句话说"、"换言之"、"更准确地说"、"本质上"
"区别于"、"不同于"、"与……相对"、"对立面是"
"可以分为"、"包括"、"包含"、"分为"、"构成"
"核心概念"、"关键概念"、"基本概念"、"重要概念"
"术语"、"行话"、"黑话"、"简称"、"缩写"
"在……中指"、"在……语境下"、"在分析中"
"反复出现"、"经常提到"、"经典论据"、"常用概念"
"所谓……"、"通常说的……"、"所谓的"
"不是……而是……"（用于纠正概念理解时）
"这个概念很重要"、"记住这个"、"关键点"
"第一章讲……"、"之前提到过……"、"之前定义过……"
"记住这个区别"、"这是核心"、"核心概念是"
"隐喻地说"、"打个比方"、"类比"、"像……一样"
```

### 上下文模式
- 主讲人用引号或语气强调某个特定词汇
- 主讲人对某个日常词汇给出非常规定义
- 同一概念在不同期视频中被反复提及并不断完善定义
- 主讲人创建了一个新的复合词或缩略语
- 某个概念被用作分析框架的核心工具
- 主讲人明确区分自己对某词的使用与常规用法的不同

## 输出格式 (Output Format)

每个提取条目使用以下 YAML 结构。类型固定为 `glossary`：

```yaml
- id: GLOS_001
  title: "[术语/概念名称]"
  type: glossary
  source_episode: "[首次出现或给出定义的视频标题/期号]"
  source_quote: "[原始定义或关键用法的引用，标注时间戳]"
  definition: |
    [100-200字的清晰定义，用主讲人的原意为核心，辅以解释性说明]
  usage_notes: |
    [50-100字的使用说明：该概念在什么语境下使用、与什么概念对照、有什么重点
     或容易混淆之处]
  related_terms:
    - "[相关概念1 — 关系描述，如 上位概念/下位概念/对立概念/平行概念]"
    - "[相关概念2]"
  tags:
    - glossary
    - [概念领域，如 game_theory / geopolitics / civilization / methodology]
    - [概念类型，如 coined_term / key_definition / analogy_framework]
    - [重要程度，如 core / secondary / contextual]
```

## 质量检查清单 (Quality Checklist)

- [ ] 该术语是否是主讲人**反复使用**的概念（而非一次性提及）？
- [ ] 是否准确捕捉了主讲人**自己给出的定义**（而非通用的百科定义）？
- [ ] 是否标注了该术语的**首次出现或给出明确定义**的视频来源？
- [ ] 是否记录了该术语与其他概念**的关联关系**（上位、下位、对立等）？
- [ ] 是否区分了**主讲人的独特用法**与常规用法的差异？

## 预期数量 (Expected Quantity)

每个系列（~30期视频）预期提取 **15–25 个** 术语条目。注意：这应该是经过筛选的精炼列表，而非每期视频中每一个新词汇的简单罗列。只有真正具有分析框架枢纽地位的术语才应被收录。
