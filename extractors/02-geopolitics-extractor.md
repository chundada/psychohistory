# 地缘政治法则提取器 (Geopolitics Extractor)

## 用途 (Purpose)

从视频转录文本中提取**地缘政治法则**相关的知识条目。识别主讲人使用或引用的地缘政治理论框架，包括陆权与海权理论、心脏地带理论、修昔底德陷阱、缓冲国理论、帝国周期、势力范围、多极/两极/单极体系的结构规律等。

## 范围 (Scope)

### 属于此提取器的内容
- 经典地缘政治理论：麦金德心脏地带理论、斯皮克曼边缘地带理论、马汉海权论、杜黑制空权理论
- 结构性国际关系理论：修昔底德陷阱、安全困境、权力转移理论、霸权稳定论
- 地理战略概念：缓冲国、势力范围、破碎地带、弧形危机带、战略纵深、内线/外线
- 帝国与文明的地缘逻辑：帝国过度扩张、边缘叛乱、文明边界、边疆假说
- 体系结构分析：单极、两极、多极体系的稳定性特征；极转移（power transition）
- 地理决定论讨论：地形、资源分布、气候、航道、海峡对战略选择的影响
- 联盟动力学：联盟形成、联盟承诺的可信度、联盟解体、中立国的地缘逻辑

### 不属于此提取器的内容
- 纯历史叙述（未提炼地缘规律）
- 国内政治选举策略（无地缘要素）
- 纯经济全球化描述（无地理战略维度）
- 军事战术细节（非战略层面）中的具体战斗流程
- 个人领导力或心理特质分析（个体层面而非结构层面）

### 上下文模式
- 地理特征被赋予战略含义（如"这个国家没有出海口所以……"）
- 国际关系被视为结构性规律而非个体决策的产物
- 对长期历史趋势的归纳，跨越不同时代和国家
- 大国与小国之间不对称的权力关系分析
- 对全球格局转换期的判断（"正在从X极走向Y极"）
- 地缘位置被用来解释国家行为或战略选择

## 输出格式 (Output Format)

每个提取条目使用以下 YAML 结构：

```yaml
- id: GEO_001
  title: "[地缘政治法则名称]"
  type: principle
  source_episode: "[视频标题/期号]"
  source_quote: "[原文引用，标注时间戳]"
  summary: |
    [200-300字的中文摘要，包含：法则的核心命题、提出者/经典来源、历史证据、适用范围、已知反例]
  tags:
    - geopolitics
    - [理论类别，如 heartland_theory / thucydides_trap / buffer_state]
    - [地理区域，如 eurasia / south_china_sea / middle_east]
  trigger_conditions:
    - [触发条件1：何种地缘配置下此法则生效]
    - [触发条件2：法则起作用所需的前提条件]
    - [触发条件3：已知的例外或失效情形]
```

## 质量检查清单 (Quality Checklist)

- [ ] 提取的法则是否具备跨历史案例的通用性（不仅是描述某一事件）？
- [ ] 是否明确引用了该法则的经典提出者或理论源头？
- [ ] 是否标注了该法则的地理适用范围（全球性/区域性）？
- [ ] 是否提供了至少一个历史案例和一个当代案例作为证据？
- [ ] 是否记录了该法则的已知局限或反例？

## 预期数量 (Expected Quantity)

每个系列（~30期视频）预期提取 **12–20 个** 地缘政治法则条目。
