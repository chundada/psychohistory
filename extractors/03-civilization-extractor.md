# 文明兴衰模式提取器 (Civilization Extractor)

## 用途 (Purpose)

从视频转录文本中提取**文明兴衰模式**相关的知识条目。识别主讲人分析社会/文明演进时使用的结构框架，包括精英过剩、人口转型、制度僵化、兴起与衰落周期、社会复杂度与崩溃的关系等宏观历史模型。

## 范围 (Scope)

### 属于此提取器的内容
- 文明生命周期理论：兴起阶段、成熟阶段、衰落阶段、崩溃阶段的标志性特征
- 精英理论分析：精英生产、精英过剩、精英循环、精英内耗、精英与大众的脱节
- 人口与社会结构：人口转型模型、马尔萨斯陷阱、生育率变化、人口结构对社会稳定的影响
- 制度分析：制度僵化、制度韧性、制度创新、寻租社会、制度化利益集团锁定
- 崩溃研究：崩溃的四大动因（环境、经济、社会、政治）、崩溃的预警信号、崩溃后的重建
- 社会复杂度：社会分工、专业化、相互依赖程度、复杂度与脆弱性的关系
- 兴衰周期规律：长周期（数百年）和短周期（数十年）的区分与特征重复
- 关键转折点分析：决定性时刻、分岔点、路径依赖、临界点

### 不属于此提取器的内容
- 单一文明不加比较的纯历史年表叙述
- 不涉及社会结构的纯文化/艺术史（如艺术风格变迁）
- 纯科技史/发明编年史（未与社会结构关联）
- 纯军事战役史（除非作为文明压力的信号）
- 微观层面的个人传记（除非作为精英分析的案例）

### 上下文模式
- A文明与B文明在相同发展阶段的行为比较
- "每个帝国都会经历……然后……最终……"这类规律性陈述
- 对当前社会趋势与历史上某个衰落期进行类比
- 使用生物学术语（生命、成长、衰老、死亡）来描述社会
- 指出社会发展达到某个极限或瓶颈后的转折
- 讨论哪些因素使某些文明更具韧性或更容易崩溃
- 跨世纪/跨千年的长时段历史分析框架

## 输出格式 (Output Format)

每个提取条目使用以下 YAML 结构：

```yaml
- id: CIV_001
  title: "[文明模式/规律名称]"
  type: principle
  source_episode: "[视频标题/期号]"
  source_quote: "[原文引用，标注时间戳]"
  summary: |
    [200-300字的中文摘要，包含：模式的核心机制、关键变量（触发条件）、典型历史案例、当代对应、模式的时间尺度]
  tags:
    - civilization
    - [模式类别，如 elite_overproduction / demographic_transition / institutional_sclerosis]
    - [涉及的文明/区域，如 rome / china / byzantium]
    - [时间尺度，如 centuries / decades]
  trigger_conditions:
    - [触发条件1：模式启动的初始条件]
    - [触发条件2：模式加速的关键信号]
    - [触发条件3：导致模式逆转的可干预因素]
```

## 质量检查清单 (Quality Checklist)

- [ ] 提取的模式是否跨越了至少两个不同文明的案例验证？
- [ ] 是否清晰定义了该模式的时间尺度（几世纪/几代人/几十年）？
- [ ] 是否区分了模式的起因、过程与结果三阶段？
- [ ] 是否标注了该模式的适用范围（适用于何种类型的社会）？
- [ ] 是否识别了该模式中的可量化指标（如精英数量、财税占比、复杂度指数）？

## 预期数量 (Expected Quantity)

每个系列（~30期视频）预期提取 **10–18 个** 文明兴衰模式条目。
